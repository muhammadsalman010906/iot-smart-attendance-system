 IoT Smart Attendance System

An RFID-based smart attendance system using ESP8266 and MFRC522 module. The system scans RFID cards and records attendance automatically.

---

Features
- RFID card scanning using MFRC522
- ESP8266 WiFi connectivity
- Automatic attendance marking
- Buzzer feedback system
- Real-time data logging

---
Hardware Used
- ESP8266 NodeMCU
- MFRC522 RFID Reader
- RFID Cards/Tags
- Buzzer
- Jumper wires
